# DK Plugins Marketplace

A Claude Code plugin marketplace with tools for testing, documentation, project management, lateral thinking, and more.

## Installation

```bash
claude plugins marketplace add divyekant/dk-marketplace
```

## Available Plugins

| Plugin | Description |
|--------|------------|
| **update-checker** | Auto-checks for updates to installed CC plugins, MCP servers, and hooks |
| **learning-skill** | Captures learnings from failures and fixes during sessions |
| **skill-conductor** | Pipeline orchestration and skill composition framework |
| **apollo** | Agent-agnostic project lifecycle manager (multi-agent support) |
| **delphi** | Comprehensive test scenario generator and executor |
| **hermes** | Audience-specific documentation generator (internal/external/marketing) |
| **think-different** | Lateral thinking with 15 reframing frameworks |
| **ui-val** | Visual UI validation at multiple viewports using multimodal vision |
| **claude-code-restart** | Self-restart mechanism with /restart command |
| **carto** | Codebase intelligence indexing and semantic search |

## Installing a Plugin

After adding this marketplace:

```bash
claude plugins install update-checker
claude plugins install apollo
# etc.
```
