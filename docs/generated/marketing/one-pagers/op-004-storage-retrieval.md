# Tiered Retrieval: The Right Context, Every Time

## The Problem

AI coding assistants face a fundamental tension with context. Give them too much information and they lose focus -- producing scattered, unfocused output. Give them too little and they guess -- missing conventions, violating boundaries, and writing code that doesn't fit.

Most tools give you one option: dump everything or dump nothing. Neither works.

## The Solution

Carto delivers context in three curated tiers, each designed for a different type of task:

- **Mini (~5KB)** -- A concise project overview. Perfect for quick lookups, simple fixes, and getting oriented in an unfamiliar codebase.
- **Standard (~50KB)** -- Detailed module-level context with patterns and relationships. The right level for everyday feature development and code reviews.
- **Full (~500KB)** -- Complete architectural analysis including cross-system relationships, design rationale, and historical context. Built for major refactors and system design work.

Each tier is curated, not truncated. Carto selects and structures the most relevant information for the level of detail you need.

## Key Benefits

- **Always the right size.** Context matched to task complexity means AI assistants stay focused and produce relevant output.
- **Millisecond retrieval.** Context is pre-built and indexed. No waiting for analysis. Ask and receive, instantly.
- **Semantic understanding.** Query in natural language. Carto understands intent, not just keywords. Ask about "error handling" and get results even if your code uses different terminology.
- **Seven layers of depth.** From individual function summaries to system-wide architectural blueprints, Carto's semantic index captures understanding at every meaningful level.

## How It Works

1. Carto pre-builds a layered semantic model of your codebase during indexing.
2. When context is requested, Carto matches the query to the most relevant parts of the model.
3. The appropriate tier delivers structured, curated context in milliseconds.

## Who It's For

Developers and AI coding assistants that need context precision. Platform teams building internal developer tools. Any team that wants AI assistants to produce code that fits their codebase on the first try.
