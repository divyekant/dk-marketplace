# External Source Integration: The Full Story Behind Your Code

## The Problem

Your codebase is only half the story. The reasons behind architectural decisions live in Jira tickets. The evolution of a feature is documented in GitHub PRs. Planned changes are outlined in Notion. Critical context from last week's debugging session lives in a Slack thread.

AI coding assistants only see the code. They miss all of this context, leading to suggestions that technically work but conflict with your team's plans, ignore known constraints, or duplicate past mistakes.

## The Solution

Carto connects to the tools your team already uses -- GitHub, Jira, Linear, Notion, Slack, and more -- and weaves that context directly into the codebase understanding it provides to AI assistants.

Now when an AI assistant works on your code, it knows not just what the code does, but why it was built that way, how it evolved, and what's changing next.

## Key Benefits

- **Complete understanding.** AI assistants that know the business context behind the code produce dramatically better suggestions and fewer false starts.
- **Automatic connections.** Carto identifies relationships between code and external sources without manual tagging or linking. Set it up once and context flows automatically.
- **Eight sources out of the box.** GitHub PRs and issues, Jira, Linear, Notion, Slack, and more. Purpose-built connectors extract the right information from each tool.
- **Signal over noise.** Carto prioritizes recent, high-impact context so AI assistants get the information that matters most for the task at hand.

## How It Works

1. **Connect** your existing tools with API keys or OAuth.
2. **Carto enriches** your codebase understanding with relevant context from every source.
3. **AI assistants receive** unified context -- code plus the "why" -- whenever they need it.

## Who It's For

Engineering teams whose development context is scattered across multiple tools. Teams that want AI assistants to understand not just the code, but the decisions, requirements, and plans that shape it.

## Get Started

Connect your tools, run an index, and your AI assistants immediately gain access to the full context behind your codebase. No manual linking. No ongoing maintenance.
