# Carto Web Dashboard: Codebase Intelligence for Everyone

## The Problem

Codebase knowledge is locked behind the command line. That means engineers can access it, but product managers, architects, and new team members are left out. They end up asking developers for context — slowing everyone down.

## The Solution

Carto includes a built-in web dashboard that gives your entire team visual access to codebase intelligence. Browse indexed projects, trigger new indexes, run queries, and configure sources — all from a browser. No CLI skills required.

## Key Benefits

**Nothing to install.** The dashboard is embedded in the Carto binary. Start the server and it's ready. No separate deployment, no frontend build, no extra infrastructure.

**Live progress tracking.** Indexing a large codebase? Watch it happen in real time. Progress indicators show exactly what's been processed and what's remaining.

**Visual queries.** Ask questions about your codebase through a clean interface. Get structured, readable answers without memorizing command flags.

**Project management.** Add and configure codebase sources for each project directly from the dashboard. No config files to edit.

**Works everywhere.** Responsive design means the dashboard works on any screen — desktop, laptop, or tablet.

## How It Works

Start the Carto server. Open your browser. The dashboard shows all your indexed projects, their status, and a query interface. Click to index a new codebase, run a query, or explore what Carto knows about your code. Everything updates in real time.

## Who It's For

Engineering leads who want codebase visibility without terminal overhead. Product managers who need to understand system architecture. Architects reviewing component relationships. New team members exploring unfamiliar codebases for the first time.

## Get Started

Launch the Carto server and navigate to the dashboard URL. Your team can start exploring codebase intelligence immediately — no training needed.
